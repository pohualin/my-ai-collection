# README Writer Skill

Create, update, or improve README.md files for any project. This skill extracts project metadata, suggests structure and content, and generates professional documentation tailored to the project type (library, app, API, etc.).

## Features
- Generate new README.md files with standard sections (Description, Installation, Usage, License, etc.)
- Extract project metadata (name, author, license, etc.)
- Suggest badges, shields, and best practices
- Write clear descriptions, usage examples, and setup instructions
- Tailor sections and content to project type or user request

## Example Usage
- "Create a README.md for my Python project."
- "Update the usage section in my README."
- "Suggest badges for my open-source repo."
