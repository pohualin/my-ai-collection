# Java Documentation Reference

For official JavaDoc documentation, see:
[Oracle JavaDoc Tool Guide](https://www.oracle.com/technical-resources/articles/java/javadoc-tool.html)

## JavaDoc Example

/**
 * Calculates the sum of two integers.
 * @param a First integer
 * @param b Second integer
 * @return Sum of a and b
 */
public int sum(int a, int b) {
    return a + b;
}

## Common JavaDoc Tags
- @param: Describes a parameter
- @return: Describes the return value
- @throws: Describes exceptions thrown
- @see: References related methods/classes
